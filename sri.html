    Smart Road Infrastructure

    Revoluciona las carreteras con Smart Road Infrastructure! Un sistema de monitorización de tráfico en tiempo real económico.

    Tecnologías usadas:

    Arduino

    MQTT

    Adafruit

    Motivación

    Este proyecto fué mi propuesta para un concurso de hackster. Decidí participar en este concurso para retarme a mí mismo y conocer la metodología de estos. A la hora de redactar mi propuesta era consciente de mi desconocimiento en ciertas tecnologías, pues nunca había trabajado con MGTT o Adafruit, pero aproveche para tomar este reto como una oportunidad.

    Smart Road Infrastructure (SRI), pretende ser una solución económica para la monitorización del tráfico. Permite obtener datos del tráfico en tiempo real gracias a sensores inductivos. En Europa existe el European Data Project, donde se organizan datos de todo tipo, entre ellos del tráfico. A nivel nacional, Madrid lidera la monitorización del tráfico. Sin embargo, otras ciudades españolas no se pueden permitir esos sistemas de monitorización. Bajo este marco decidí crear un sistema realmente económico, que puedan permitirse ciudades con menos recursos, y por encima de todo que trate los datos de forma anónima y segura.

    • European data project: https://data.europa.eu/en
    • Madrid tráfico en tiempo real: https://informo.madrid.es/#/realtime?panel=live

    Step-by-step

    Empezando por el hardware, el circuito debe seguir el siguiente diagrama:

    En mi caso yo he utilizado una fuente de tensión regulable

    Dado que este proyecto se trata de un prototipo, las medidas se harán con un coche de juguete, pero dado que la parte inferior de este es de plástico será necesario pegar un poco de papel de aluminio para que el sensor inductivo funcione.

    Ahora vamos con el software. Para usar la conexión LTE, en el caso de usar la tarjeta SIM que incluye nuestra placa, es necesario registrarse en la web de Truphone y activar la tarjeta.

    Lo siguiente será actualizar el firmware, en este paso tienes que conectar tu placa al ordenador con un USB tipo C, pero ten en cuenta que el cable debe permitir el paso de datos, pues hay cables tipo C que solo permiten carga de dispositivos, y si se usa uno de estos cables nuestro ordenador no detectará la placa. Una vez conectada podemos descargarnos el firmware de la página del fabricante.

    Antes de subir el código vamos a configurar el servidor de Adafruit. El primer paso es crearse una cuenta si aún no tienes una. Una vez registrado accede al menú IO, y a continuación a dashboard. Aquí puedes personalizar el dashboard en función de tus preferencias. En mi caso he decidido poner un log para monitorizar en todo momento el funcionamiento del sistema. En segundo lugar un elemento gráfica para mostrar el número de coches en función del tiempo. Por último un bloque que muestre la temperatura por si se detectara una subida preocupante de esta parar el sistema.

    Antes de subir el código utilizando el IDE de Arduino debes realizar unos pequeños ajustes. En las primeras líneas de código debes poner tanto tu usuario de Adafruit, como la key del servidor que has creado. Esta key se genera de forma automática, no es la contraseña de tu cuenta.

    Ahora toca configurar el IDE de Arduino para poder cargar el código en la placa.
